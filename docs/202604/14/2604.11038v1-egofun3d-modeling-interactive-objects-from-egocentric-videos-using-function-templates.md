## 摘要
我们提出了 EgoFun3D，这是一种针对从自我中心视频中建模交互式三维物体的协调任务定义、数据集和基准。交互式物体对于具身智能（embodied AI）具有重要意义，但相对稀缺，因此能够从现成的真实世界视频中进行建模具有很高的价值。我们的任务专注于从自我中心视频输入中获取可用于仿真的交互式三维物体。虽然先前的研究主要集中于关节运动，但我们通过“功能模板”（function templates）——一种结构化的计算表示——捕捉了跨部件的一般功能映射（例如，炉灶旋钮的旋转控制炉灶燃烧器的温度）。功能模板使得精准评估成为可能，并能够直接编译为可在不同仿真平台上执行的代码。为了实现全面的基准测试，我们引入了一个包含 271 段自我中心视频的数据集，这些视频展示了具有挑战性的真实世界交互，并配有三维几何、二维与三维分割、关节运动及功能模板注释。为解决这一任务，我们提出了一个由四个阶段组成的处理流程：二维部件分割、重建、关节估计以及功能模板推理。全面的基准测试结果表明，该任务对现有通用方法而言具有相当的挑战性，强调了未来研究的潜在方向。

## Abstract
We present EgoFun3D, a coordinated task formulation, dataset, and benchmark for modeling interactive 3D objects from egocentric videos. Interactive objects are of high interest for embodied AI but scarce, making modeling from readily available real-world videos valuable. Our task focuses on obtaining simulation-ready interactive 3D objects from egocentric video input. While prior work largely focuses on articulations, we capture general cross-part functional mappings (e.g., rotation of stove knob controls stove burner temperature) through function templates, a structured computational representation. Function templates enable precise evaluation and direct compilation into executable code across simulation platforms. To enable comprehensive benchmarking, we introduce a dataset of 271 egocentric videos featuring challenging real-world interactions with paired 3D geometry, segmentation over 2D and 3D, articulation and function template annotations. To tackle the task, we propose a 4-stage pipeline consisting of: 2D part segmentation, reconstruction, articulation estimation, and function template inference. Comprehensive benchmarking shows that the task is challenging for off-the-shelf methods, highlighting avenues for future work.

## 二、方法论与技术方案

### 核心思想
- 提出一种新的 **功能模板（Function Template）** 表示法，将物体的“功能”形式化为 **部件之间的映射与物理效应**。
- 定义“受体（Receptor）”与“效应器（Effector）”的关系：受体是被人操作的部件，效应器是随之产生变化的部件。  
  - 用形式化表达：  
    \[
    s_E = F(s_R) = M_P(s_R)
    \]
    其中 \( M \) 为映射关系，\( P \) 为物理效应。

### 功能模板结构
- **映射类型** (\(M\))：二值（binary）、阶跃（step）、线性（linear）、累积（cumulative）。  
- **物理效应** (\(P\))：几何变化（geometry）、照明变化（illumination）、温度变化（temperature）、流体变化（fluid）。  
- 此抽象可直接编译为不同物理仿真平台的执行代码（如 Genesis、Isaac Sim、BEHAVIOR）。

### 建模流程（四阶段管线）
1. **二维部件分割（Segmentation）**：通过视觉语言模型（VLM）识别受体和效应器，调用如 SAM3 进行语义分割。  
2. **三维重建（Reconstruction）**：基于视频中的多视角帧重构物体几何。使用例如 Depth Anything 3、MapAnything、ViPE 等模型。  
3. **关节估计（Articulation Estimation）**：使用 iTACO 和 ArtiPoint 估计关节类型、轴向、起点和运动范围。  
4. **功能模板推理（Function Template Inference）**：由大规模视觉语言模型（如 Gemini 3 Flash、GPT-5 mini）推理物理效应类型与数值映射关系。

最后，各子模块输出的结果被编译成可直接用于仿真平台的执行代码，使物体具备交互性。

### 数据集构建
- **新数据集：EgoFun3D**

- 相比其他数据集（SceneFun3D、ArtVIP、Hoi!、Arti4D 等），EgoFun3D 的 **标注维度最完整（4 模态×4 注释类型）**。

### benchmark 设置
- 对比的“现成”方法包括：
  - **分割阶段**：SAM3 & Qwen3-VL、Sa2VA、X-SAM、VisionReasoner 等。
  - **重建阶段**：Depth Anything 3、MapAnything、ViPE。
  - **关节估计阶段**：iTACO、ArtiPoint。

- 每模块分别用IoU、Chamfer距离、角度/位置误差、分类准确率等作为评估指标。

## 六、主要结论与发现

- **难点识别**：
  - 二维分割和对小部件的识别是最大瓶颈；
  - 三维重建容易失真、对动态视频极度敏感；
  - 关节参数预测极度不稳定；
  - 相较之下，VLM 的功能模板推理效果较好。
- **方法有效性**：
  - 功能模板在表示与仿真兼容性上表现突出，可跨平台编译执行。
- **数据贡献**：
  - 提供首个涵盖视频、几何、关节与功能全注释的 **真实交互式物体数据集**。
