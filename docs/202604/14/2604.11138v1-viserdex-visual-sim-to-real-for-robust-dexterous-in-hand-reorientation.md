## 摘要
手内物体重定位需要对物体姿态进行精确估计，以应对复杂的任务动力学。尽管 RGB 感知为姿态跟踪提供了丰富的语义线索，现有方法通常依赖多摄像头设置或昂贵的光线追踪计算。我们提出了一种基于单目 RGB 的手内重定位仿真到现实框架，该框架引入了三维高斯喷溅（3D Gaussian Splatting, 3DGS）以弥合视觉仿真与现实之间的差距。我们的关键见解是，在高斯表示空间中执行域随机化：通过对 3D 高斯进行物理一致的预渲染增强，我们生成用于物体姿态估计的高逼真度、随机化视觉数据。操控策略通过基于课程的强化学习结合教师-学生蒸馏进行训练，从而高效学习复杂的行为。重要的是，感知与控制模型均可在消费级硬件上独立训练，无需大型计算集群。实验结果表明，使用 3DGS 数据训练的姿态估计器在具有挑战性的视觉环境中优于采用传统渲染数据训练的模型。我们在配备 RGB 摄像头的物理多指机械手上验证了该系统，展示了在具有挑战光照条件下对五种不同物体的稳健重定位。我们的结果表明，高斯喷溅为仅基于 RGB 的灵巧操控提供了切实可行的途径。有关硬件部署视频及更多补充材料，请参见项目网站：https://rffr.leggedrobotics.com/works/viserdex/

## Abstract
In-hand object reorientation requires precise estimation of the object pose to handle complex task dynamics. While RGB sensing offers rich semantic cues for pose tracking, existing solutions rely on multi-camera setups or costly ray tracing. We present a sim-to-real framework for monocular RGB in-hand reorientation that integrates 3D Gaussian Splatting (3DGS) to bridge the visual sim-to-real gap. Our key insight is performing domain randomization in the Gaussian representation space: by applying physically consistent, pre-rendering augmentations to 3D Gaussians, we generate photorealistic, randomized visual data for object pose estimation. The manipulation policy is trained using curriculum-based reinforcement learning with teacher-student distillation, enabling efficient learning of complex behaviors. Importantly, both perception and control models can be trained independently on consumer-grade hardware, eliminating the need for large compute clusters. Experiments show that the pose estimator trained with 3DGS data outperforms those trained using conventional rendering data in challenging visual environments. We validate the system on a physical multi-fingered hand equipped with an RGB camera, demonstrating robust reorientation of five diverse objects even under challenging lighting conditions. Our results highlight Gaussian splatting as a practical path for RGB-only dexterous manipulation. ## 一、研究核心问题与整体动机

- **核心问题**：如何在单目 RGB 视觉条件下实现灵巧机械手对复杂形状物体的稳定手内重定位（In-hand Reorientation）。
- **难点背景**：
  - 视觉难题：在手内操作中，物体经常被手指遮挡，导致姿态估计误差大；
  - 仿真与现实差距（Sim-to-Real Gap）：传统基于网格渲染或光线追踪的视觉数据生成开销高，难以大规模训练；
  - 控制难题：端到端强化学习需要同时学习感知、估计、控制三者，样本效率低，收敛慢。
- **研究动机**：
  - 希望通过集成高效的视觉仿真与分阶段训练机制，使仅凭消费级硬件即可训练出鲁棒的视觉操控策略；
  - 利用**3D Gaussian Splatting (3DGS)**，在保持光照一致性与物理逼真的同时，降低渲染计算成本。

## 二、方法论与技术框架

### 1. 核心思想

ViserDex 提出一个**分阶段的视觉-控制联合学习框架**，实现从仿真到现实的单目 RGB 灵巧手控制。  
整体流程分为四个阶段：

1. **教师策略训练（Teacher Policy）**：在仿真中使用完全状态信息，通过强化学习（PPO）学习准确的操控策略；
2. **学生蒸馏（Student Policy）**：将教师的行为蒸馏到一个仅依赖噪声观测的循环神经网络学生模型；
3. **视觉姿态估计器训练**：基于 3D 高斯溅射生成的虚拟数据训练单目关键点预测网络；
4. **真实部署（Deployment）**：在真实机械手上利用 RGB 相机估计物体位姿，输入给学生策略执行实时控制。

### 2. 仿真与渲染创新：Pre-rasterization Augmentation

- 引入 **3D 高斯溅射（3DGS）** 作为显式场景表示，避免传统网格渲染的高开销；
- 在渲染前，对高斯参数（位置、颜色、球谐系数）进行**预渲染增广（Pre-rasterization Augmentations）**；
- 四类增广策略：
  1. **Random Noise**：高频随机扰动，模拟传感噪声；
  2. **Spatial Cluster**：基于空间聚类，模拟局部阴影与脏污；
  3. **Color Cluster**：基于材质相关颜色聚类，模拟反射与材质差异；
  4. **Global Shift**：全局亮度、色温、曝光等变化，模拟光照变化；
- 这些增广在高斯空间执行，确保物理一致性且渲染代价低（<4% 总耗时）。

### 3. 强化学习策略细节

- **算法**：PPO（Proximal Policy Optimization）。
- **观测拆分**：
  - Proprioceptive（机器人关节信息）；
  - Exteroceptive（物体姿态）；
  - Privileged（仅仿真可得的完整状态）。
- **课程学习机制（Performance-based Curriculum）**：
  - 随性能增加逐步提高任务难度；
  - 包括动作延迟、时间窗口约束、正则化项自适应调整；
- **学生蒸馏**：
  - 使用行为克隆与重建损失；
  - 加入观测噪声模型（延迟、抖动、随机丢帧等）；
  - 在线 DAgger 策略缓解状态分布偏移。

### 4. 视觉姿态估计算法

- **架构**：ResNet-34 主干 + 关键点回归头；
- **输出**：九个关键点（八个几何点 + 一个中心点）的 2.5D 坐标；
- **姿态解算**：采用 **Rigid Procrustes 算法** 估计 6D 姿态；
- **训练数据**：由教师策略在仿真中的轨迹通过 3DGS 渲染生成，含随机噪声与运动模糊增强。

## 三、实验设计与对比设置

### 1. 实验场景与硬件
- 实体平台：**16 自由度 Allegro Hand** + **Intel RealSense D435i 单目 RGB 摄像头**；
- 对象集合：5 种物体（Cube、Globe、3D Printed Toy、Tablet Bottle、Rubber Duck）；
- 光照条件：
  - **Nominal**：标准白光；
  - **Adversarial**：多彩点光源、低亮度、动态色温变化。

### 2. 对比方法（Pose Estimation）
1. 标准 Tiled 渲染；
2. 随机场景随机化 (Domain Randomized, DR)；
3. 朴素高斯溅射 (Naïve GS)；
4. 本文方法（带预渲染增强的 3DGS）。

### 3. Benchmark 与指标
- **ADD（Average Distance of Model Points）**；
- **<10mm & <10° 准确率**；
- **硬件评估指标**：物体平均连续成功次数（Consecutive Success, CS）。

### 4. 消融实验
- 对四种增广方式分别“去除一项”评估；
- 分析课程学习策略三部分（Penalty、Latency、Time Window）；

- 评估感知噪声处理及信念状态过滤（belief decoder）。
