## 摘要
在非结构化环境中，实现具备功能性的灵巧抓取需要语义理解、精确的三维功能定位与可物理解释的执行紧密结合。相比端到端的 VLA 方法，模块化层次方法更具可控性与可解释性，但现有方法仍依赖预定义的可供性标签，且缺乏实现功能性灵巧操控所需的语义—姿态紧密耦合。为此，我们提出 BLaDA（Bridging Language to Dexterous Actions in 3DGS fields），这是一个可解释的零样本框架，将开放词汇的指令转化为功能性灵巧操控的感知与控制约束。BLaDA 首先通过知识引导的语言解析（KLP，Knowledge-guided Language Parsing）模块，将自然语言解析为结构化的六元操作约束，建立了可解释的推理链。为实现姿态一致的空间推理，我们引入了三角功能点定位（TriLocation）模块，利用 3D Gaussian Splatting 作为连续场景表示，并在三角几何约束下识别功能区域。最后，3D 关键点抓取矩阵变换执行（KGT3D+）模块将这些语义—几何约束解码为可物理实现的腕部姿态与手指层级指令。在复杂基准上的大量实验表明，BLaDA 在可供性定位精度与跨多类别、多任务的功能性操作成功率上均显著优于现有方法。代码将公开于 https://github.com/PopeyePxx/BLaDA。

## Abstract
In unstructured environments, functional dexterous grasping calls for the tight integration of semantic understanding, precise 3D functional localization, and physically interpretable execution. Modular hierarchical methods are more controllable and interpretable than end-to-end VLA approaches, but existing ones still rely on predefined affordance labels and lack the tight semantic--pose coupling needed for functional dexterous manipulation. To address this, we propose BLaDA (Bridging Language to Dexterous Actions in 3DGS fields), an interpretable zero-shot framework that grounds open-vocabulary instructions as perceptual and control constraints for functional dexterous manipulation. BLaDA establishes an interpretable reasoning chain by first parsing natural language into a structured sextuple of manipulation constraints via a Knowledge-guided Language Parsing (KLP) module. To achieve pose-consistent spatial reasoning, we introduce the Triangular Functional Point Localization (TriLocation) module, which utilizes 3D Gaussian Splatting as a continuous scene representation and identifies functional regions under triangular geometric constraints. Finally, the 3D Keypoint Grasp Matrix Transformation Execution (KGT3D+) module decodes these semantic-geometric constraints into physically plausible wrist poses and finger-level commands. Extensive experiments on complex benchmarks demonstrate that BLaDA significantly outperforms existing methods in both affordance grounding precision and the success rate of functional manipulation across diverse categories and tasks. Code will be publicly available at https://github.com/PopeyePxx/BLaDA.

## 二、方法论与核心技术

### 1. 整体框架

BLaDA（**Bridging Language to Dexterous Actions in 3DGS Fields**）采用三阶段模块化流水线结构：

1. **KLP（Knowledge-guided Language Parsing）** – 知识引导的语言解析；
2. **TriLocation（Triangular Functional Point Localization）** – 三角功能点定位；
3. **KGT3D+（Keypoint Grasp Matrix Transformation Execution）** – 关键点抓取矩阵变换与执行。

整个流程的目标是：  
自然语言指令 → 六元结构约束 → 三维功能区域 → 物理控制命令。

---

### 2. KLP 模块：语言到结构约束的解析

- 将自然语言指令解析为结构化六元组  
  \( S = (g_a, g_r, g_t, g_f, \tau, \kappa) \)，其中：
  - \( g_a \)：抓取可供性（目标部件区域）
  - \( g_r \)：手指角色分配（功能型手指）
  - \( g_t \)：抓取类型（姿势、姿态类型）
  - \( g_f \)：抓取力
  - \( \tau \)：工具拓扑结构（如杆状、柄状、旋钮、平面）
  - \( \kappa \)：任务意图（如 press、click、open、hold）
- 通过结合**大型语言模型 (LLM)** 与 **结构化知识图谱（F2F knowledge base）**，进行语义到物理约束的零样本映射。
- 引入分类提示（Prompt），结合上下文示例与功能性词汇聚类，使解析具有物理执行一致性。

### 3. TriLocation 模块：基于 3DGS 的功能点定位

- **核心思想**：在 3D Gaussian Splatting (3DGS) 场中，将功能区域表示为连续的高斯点云场，实现语义与几何的统一。
- 关键步骤：
  1. **构建层级语义场（HSE）**：利用 YOLO 检测与 SAM 分割生成对象级和部件级掩膜；通过“选择”与“上下文裁剪”策略避免语义漂移；
  2. **计算相似度**：利用 CLIP 提取语言语义向量，与 3DGS 中的特征点相似度匹配得到第一个关键点 \(p_1\)；
  3. **预测相对偏移**：轻量 MLP 基于语义约束预测两个偏移点 \(p_2, p_3\)；
  4. 三点形成**抓取几何三角结构 {p1, p2, p3}**；
  5. **构建局部坐标系**：结合任务意图 κ 与工具拓扑 τ ，经 PCA 与 RANSAC 算法提取结构轴、表面法向或旋转轴，定义右手坐标系 (x̂, ŷ, ẑ)。

- 三点结构以 **三角形几何约束**约束手部姿态与物理空间一致性，从而增强语义-姿态耦合。

---

### 4. KGT3D+ 模块：三维矩阵变换与执行

- 接收三点 \(p_1, p_2, p_3\) 后，构造手掌的位姿矩阵：
  \[
  R = [\vec{x}, \vec{y}, \vec{z}], \quad T = p_3
  \]
  其中  
  \(\vec{z} = p_1 - p_3\)，  
  \(\vec{y} = (p_2 - p_3) \times \vec{z}\)，  
  \(\vec{x} = \vec{y} \times \vec{z}\)。
- 根据语言解析得到的 \(g_t\)（抓取类型）与 \(g_f\)（抓取力），在 **F2F 功能性抓取库**中查找对应的手指关节角度 \(J\) 与力分布 \(F\)，生成可物理执行的动作指令。
